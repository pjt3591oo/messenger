# 설명
해당 프로젝트는 vuejs를 연습하기 위해 메신저 프로그램인 카톡을 모방하여 만드는 프로젝트입니다
css를 대충 입혀서 매우 닮지는 않았습니다

# 사용 된 프레임워크

```
client framework : vuejs
server framework : express
database : 미정
```

# 실행

## 1 서버 의존모듈 설치 및 실행
```
$ cd server 
$ npm install
$ node ./bin/www
```

## 2 클라이언트 모듈 설치 및 실행
```
$ cd client/message
$ npm install
$ npm run dev
```
